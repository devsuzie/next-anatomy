### 경고 :warning:

모든 프로젝트의 환경 설정은 동일하다. clone 받은 후 터미널에 아래와 같이 명령어
를 입력해라. 포트는 3000이다.

```
// 실행하기를 원하는 프로젝트로 이동
cd ${project-name}

// 실행
npm install
npm run dev
```
